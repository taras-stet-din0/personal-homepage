# Homepage
A responsive personal homepage built with HTML, CSS, and JavaScript to present information about me, my skills, projects, and contact details. 

🔗 Live demo: [https://taras-stet-din0.github.io/personal-homepage/](link)

!Screenshot <img width="1893" height="858" alt="image" src="https://github.com/user-attachments/assets/1d2d6ebe-2937-460b-a362-f6b549c32c88" />


## Features
- Responsive personal portfolio layout 
- Sections for personal information, skills, and projects
- Clean and simple UI design
- Navigation between page sections
- Mobile-friendly layout

## Tech stack
- HTML5
- CSS3 (Flexbox/Grid)
- JavaScript (ES6)

## How to run locally
1. git clone https://github.com/taras-stet-din0/personal-homepage.git
2. Open index.html in your browser

## What I learned
This project helped me practice building responsive layouts, organizing page structure, styling components with CSS, and improving overall frontend development skills.
